# SmartNewsBot - Умный Telegram-бот для фильтрации новостей

Telegram-бот на aiogram 3, который собирает новости из заданных источников и фильтрует их по сложным смысловым темам с использованием Google Gemini API.

## 📋 Требования

- Python 3.10 или выше
- Telegram Bot Token (от @BotFather)
- Google Gemini API Key (от Google AI Studio)

## 🚀 Пошаговая инструкция по запуску

### Шаг 1: Проверка версии Python

Откройте терминал (PowerShell на Windows) и проверьте версию Python:

```powershell
python --version
```

Должно быть **Python 3.10** или выше. Если Python не установлен или версия ниже, скачайте с [python.org](https://www.python.org/downloads/).

### Шаг 2: Создание виртуального окружения (опционально, но рекомендуется)

Виртуальное окружение изолирует зависимости проекта от системных. Это хорошая практика, но не обязательно.

**Вариант А: С виртуальным окружением (рекомендуется)**

```powershell
# Перейдите в папку проекта
cd "g:\Мой диск\Программы\SmartNewsBot"

# Создайте виртуальное окружение
python -m venv venv

# Активируйте виртуальное окружение
.\venv\Scripts\Activate.ps1
```

После активации в начале строки терминала появится `(venv)`.

**Если появляется ошибка при активации:**

1. **Ошибка политики выполнения скриптов:**
   ```powershell
   Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
   ```
   Затем попробуйте активировать снова.

2. **Файл не найден:** Убедитесь, что вы находитесь в правильной папке проекта и что команда `python -m venv venv` выполнилась успешно.

3. **Альтернативный способ активации (если PowerShell не работает):**
   ```cmd
   venv\Scripts\activate.bat
   ```

**Вариант Б: Без виртуального окружения**

Если не хотите использовать виртуальное окружение, просто пропустите этот шаг и установите зависимости напрямую (см. Шаг 3).

### Шаг 3: Установка зависимостей

Установите все необходимые библиотеки:

```powershell
pip install -r requirements.txt
```

Это установит:
- `aiogram` - фреймворк для Telegram ботов
- `aiosqlite` - асинхронная работа с SQLite
- `APScheduler` - планировщик задач
- `python-dotenv` - загрузка переменных окружения
- `google-generativeai` - Google Gemini API

### Шаг 4: Настройка переменных окружения

Откройте файл `.env` в редакторе и убедитесь, что указаны оба ключа:

```env
# Telegram Bot Token (получить у @BotFather)
BOT_TOKEN=ваш_токен_бота

# Google Gemini API Key (получить на https://aistudio.google.com/app/apikey)
GEMINI_API_KEY=ваш_gemini_ключ
```

#### Как получить Telegram Bot Token:

1. Откройте Telegram и найдите бота [@BotFather](https://t.me/BotFather)
2. Отправьте команду `/newbot`
3. Следуйте инструкциям: укажите имя и username бота
4. BotFather пришлет токен вида: `1234567890:ABCdefGHIjklMNOpqrsTUVwxyz`
5. Скопируйте токен в `.env` файл

#### Как получить Gemini API Key:

1. Перейдите на [Google AI Studio](https://aistudio.google.com/app/apikey)
2. Войдите в свой Google аккаунт
3. Нажмите "Create API Key" или "Get API Key"
4. Скопируйте ключ (начинается с `AIza...`)
5. Вставьте ключ в `.env` файл

**Важно:** Не публикуйте файл `.env` в публичных репозиториях! Он содержит секретные ключи.

### Шаг 5: Запуск бота

Запустите бота командой:

```powershell
python main.py
```

Вы должны увидеть сообщения в консоли:
```
INFO:__main__:Starting bot polling...
INFO:brain:AIAnalyzer initialized with Gemini API
```

### Шаг 6: Проверка работы бота

1. Откройте Telegram
2. Найдите вашего бота по username, который вы указали при создании
3. Отправьте команду `/start`
4. Бот должен ответить приветственным сообщением

## 📱 Использование бота

### Основные команды:

- `/start` - Регистрация и приветствие
- `/add_source` - Добавить источник новостей (сайт или Telegram-канал)
- `/add_topic` - Добавить тему для фильтрации новостей
- `/my_subs` - Показать список ваших источников и тем

### Пример работы:

1. **Добавьте источник:**
   ```
   /add_source
   ```
   Бот попросит ссылку. Отправьте, например:
   ```
   https://rss.newsru.com/rss/news
   ```
   Затем выберите тип источника (Сайт или Канал)

2. **Добавьте тему:**
   ```
   /add_topic
   ```
   Опишите тему, например:
   ```
   Изменение ключевой ставки ЦБ РФ
   ```
   или
   ```
   Теннис турниры Большого шлема начиная с 1/4 финала
   ```

3. **Проверьте подписки:**
   ```
   /my_subs
   ```

4. Бот будет ежедневно в 9:00 UTC проверять новости из ваших источников и отправлять релевантные статьи с кнопками для оценки (👍, 👎, 🔗 Читать).

## ⚠️ Устранение проблем

### Ошибка: "BOT_TOKEN is not set"
- Проверьте, что файл `.env` существует в корне проекта
- Убедитесь, что в `.env` есть строка `BOT_TOKEN=...`
- Проверьте, что нет лишних пробелов вокруг `=`

### Ошибка: "GEMINI_API_KEY не установлен"
- Проверьте наличие `GEMINI_API_KEY=...` в `.env`
- Убедитесь, что ключ скопирован полностью

### Ошибка: "ModuleNotFoundError"
- Убедитесь, что виртуальное окружение активировано
- Выполните `pip install -r requirements.txt` еще раз

### Бот не отвечает на команды
- Проверьте, что бот запущен (нет ошибок в консоли)
- Убедитесь, что вы пишете правильному боту
- Проверьте логи в консоли на наличие ошибок

### Ошибки подключения к API
- Проверьте интернет-соединение
- Убедитесь, что API ключи валидны
- Для Gemini: проверьте квоты на [Google AI Studio](https://aistudio.google.com/app/apikey)

## 📝 Примечания

- База данных SQLite создается автоматически при первом запуске (`smartnewsbot.db`)
- Планировщик настроен на отправку новостей ежедневно в 9:00 UTC
- Для работы парсеров сайтов и Telegram-каналов нужно будет реализовать функции в `parser.py`
- Бот обучается на ваших реакциях (лайки/дизлайки) для улучшения фильтрации

## 🔐 Безопасность

- **Никогда не публикуйте файл `.env`** в публичных репозиториях
- Добавьте `.env` в `.gitignore`, если используете Git
- Храните API ключи в безопасности
- При компрометации ключа - немедленно отзовите его и создайте новый
